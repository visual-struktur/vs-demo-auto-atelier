# VS Demo — Auto Atelier (Premium One-Page)

Dieses Repository enthält eine **Premium-Demo-One-Page** (Auto-Werkstatt), erstellt als Referenzprojekt für **Visual Struktur**.  
Ziel: In kurzer Zeit die Qualität von **Design, Struktur, Karten-UI, CTA-Logik und sauberer Frontend-Umsetzung** zu demonstrieren.

---

## Projektziel

- Premium-Look (Dark / Glass / High Contrast)
- Klare Nutzerführung (Hero → Leistungen → Vertrauen → Preise → Kontakt)
- Saubere, wartbare Struktur (HTML/CSS/JS getrennt)
- Geeignet für **GitHub Pages** (statisches Hosting ohne Build-Prozess)
